名称 类型 描述
code str 可转债代码
name str 可转债名称
stock_code str 正股代码
stock_name str 正股名称
freq int/BarFreq k线周期
start date 开始时间，默认全部
end date 结束时间，默认全部
skip_rt bool 是否忽略实时行情的数据
